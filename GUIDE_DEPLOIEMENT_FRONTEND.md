## 🎯 ÉTAPE 1 : CRÉER UN DOSSIER

**Sur votre ordinateur, créez :**

```

```

**Ou tapez en Terminal :**

```bash

cd frontend-livraison-cotonou
```

## 📥 ÉTAPE 2 : METTRE LES FICHIERS

**Téléchargez les 7 fichiers et mettez-les TOUS dans le dossier :**

```
├── vite.config.js
├── index.html
```

**✓ Vérifiez que TOUS les fichiers sont là !**

## 🔧 ÉTAPE 3 : INITIALISER GIT

**Dans le Terminal, allez dans le dossier :**

```bash

```

**Puis tapez :**

```bash

```

**Puis :**

```bash

```

**Puis :**


## 🎓 SI BLOQUÉ

**Problème :** "Dossier pas trouvé"
**Solution :** `cd` dans le bon dossier, puis `git init`

**Problème :** "GitHub repo existe pas"
**Solution :** Allez à https://github.com/new et créez-le d'abord

**Problème :** "Push échoue"
**Solution :** Vérifiez que votre email GitHub est correct dans votre terminal

**Problème :** "Vercel deploy échoue"
**Solution :** Attendez 5 min et rechargez la page Vercel
