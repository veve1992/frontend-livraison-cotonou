## 🚀 DÉPLOIEMENT RAPIDE (10 MINUTES)

### **ÉTAPE 1 : Créer un dossier pour le Frontend**

```bash
mkdir frontend-livraison
cd frontend-livraison
```

### **ÉTAPE 2 : Copier les fichiers**

Copiez ces 5 fichiers dans le dossier :
- `package.json`
- `main.jsx`
- `App.jsx`
- `App.css`
- `index.html`
- `vite.config.js`

### **ÉTAPE 3 : Initialiser Git**

```bash
git init
git add .
git commit -m "Frontend React Livraison Cotonou"
```

### **ÉTAPE 4 : Push sur GitHub**

1. Créez un repo GitHub : `frontend-livraison-cotonou`
2. Connectez votre repo local :

```bash
git remote add origin https://github.com/veve1992/frontend-livraison-cotonou.git
git branch -M main
git push -u origin main
```

### **ÉTAPE 5 : Déployer sur Vercel**

1. Allez : **https://vercel.com**
2. Cliquez : **"Add New"** → **"Project"**
3. Sélectionnez : **"frontend-livraison-cotonou"**
4. Cliquez : **"Deploy"**

**Attendez 2 minutes...**

### **ÉTAPE 6 : Votre URL**

```
https://frontend-livraison-cotonou.vercel.app
```

**✅ C'EST PRÊT ! 🎉**

## 🔧 DÉVELOPPEMENT LOCAL

### **Installer les dépendances**

```bash
npm install
```

### **Lancer le serveur local**

```bash
npm run dev
```

**Ouvrez :** http://localhost:3000

### **Builder pour production**

```bash
npm run build
```

## 🌐 CONNEXION À L'API

Le Frontend se connecte automatiquement à :

```
https://saas-livraison-cotonou.vercel.app
```

**Endpoints utilisés :**
- `GET /parcels` → Liste des colis
- `GET /livreurs` → Liste des livreurs
- `GET /stats` → Statistiques
